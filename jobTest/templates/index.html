Тестовое задание:

Сделать минисайт на django. Сайт содержит упрощенный каталог кинофестивалей. Фестиваль описывается следующими полями:

  • название
  • страна - текстовое поле
  • дата начала

Сайт содержит одну страницу со списком кинофестивалей. Для кинофестивалей выводятся все имеющиеся поля. Плюс есть возможность изменить сортировку фестивалей: по названию или по дате проведения. Так же список фестивалей можно отфильтровать по названию. Фильтрация и сортировка должны работать без перезагрузки страницы. Дизайн не важен.
